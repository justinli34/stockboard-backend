# StockBoard Backend

Stock data API

## Prerequisites

- uv

## Installation

1. Clone the repo

2. Install dependencies:

    ```
    uv sync
    ```

3. Start the server:

    ```
    uv run start
    ```

## Scripts

- `uv run start`: Start the server
- `make dev`: Start server with auto-reload
- `make check`: Run type checker and linter
- `make format`: Run formatter
